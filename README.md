# VitaSphere - Lifestyle Magazine Blog

A comprehensive lifestyle blog website featuring magazine-style design with content across six categories: Fashion & Accessories, Health & Beauty, Home & Garden, Travel & Accommodation, Finance & Insurance, and Food & Beverage.

## Features

- **Magazine/Editorial Design**: Clean, typography-focused layout with cold color palette
- **6 Content Categories**: Comprehensive coverage of lifestyle topics
- **Blog Articles**: 5 detailed articles with dates from January to August 2025
- **Product Reviews**: Curated product recommendations with ratings
- **Search Functionality**: Search articles, products, and reviews
- **Category Filtering**: Filter content by category
- **Pagination**: Navigate through articles with pagination
- **Responsive Design**: Mobile-friendly layout
- **Social Media Integration**: Social media icons widget
- **Smooth Animations**: CSS transitions and animations
- **Static Website**: Pure HTML, CSS, and JavaScript - no build process required

## Usage

1. Open `index.html` in a web browser
2. Navigate through categories using the category cards
3. Use the search bar to find specific content
4. Click on articles to read full content
5. Visit the Products page to see product recommendations
6. Use the contact form to send messages

## Customization

To add new articles, edit `js/data.js` and add entries to the `blogArticles` array.

To add new products, edit `js/data.js` and add entries to the `products` array.

To modify styling, edit `css/style.css`.

## Notes

- This is a static website - no server-side processing required
- Images are loaded from Unsplash (external CDN)
- All functionality is client-side JavaScript
- No build process or dependencies required
